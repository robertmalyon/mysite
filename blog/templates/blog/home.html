    Brighton · UK · rob@rob-blog.dev

    In my last year at primary school, a "Personal Computer" appeared. I was fascinated. As part of an elite team of like-minded individuals (nerds, some might say), we set out to master this new technology - as well as being responsible for removing and replacing the dust cover at the start and end of each day! By the time I left college, computers were everywhere.

    I enjoyed a 20-year long career in retail, culminating in owning my a video-game shop specializing in retro games, Ninja Game Den.

    I've also immersed myself in learning to code with an ongoing goal of becoming a full-stack web developer. Now in my 30's I still say that's what I want to be when I grow up!

    This is a live demonstration of my experience and learning, in particular the self-directed kind, which I don't have certificates for. And it's also a learning process, undergoing continued development as I discover new technologies and gain new skills. Built using Python's Django Web-Framework, currently, there's more going on behind the scenes than there is out front!

    If you're learning to code too, then check out my blog where I'll recommend lots of resources that have helped me. If you want to know if I can help you or your organisation then read on or get in touch!
